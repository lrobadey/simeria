My name is Luca. This is an intro to a a simulation project I'm building. Essentially, I'm trying to create a persistent world with agents that can take actions to build settlements, farm, trade, craft things, and create a world. The user is essentially hands-off - the goal of this simulation is to build a world these deterministic agents (like Rimworld or Dwarf Fortress) can take actions in and explore. Emergent complexity from the interaction between very simple systems is the goal. Systems thinking FTW. Ideas I have for this game include a memory system for agents as they discover the world - as they explore, they'll learn about the land and what it holds. This memory can fade over time if they haven't visited that area recently. They can farm, and raise animals, and build houses and boats.
